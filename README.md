# Create Roar App

Create a ROAR app with one simple command.

## Usage

```bash
npx create-roar-app <name>
```

And then follow the onscreen prompts. For further details, see the [ROAR Developer's Guide](https://yeatmanlab.github.io/create-roar-app/).
